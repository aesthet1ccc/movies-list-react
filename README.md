# Тестовое задание VK

> Шаги для запуска приложения:

1. Установка зависимостей

   ```
   npm install
   ```

2. Создайте .env файл в корне проекта локально
3. Cкопируйте токен для взаимодействия с API из .env.default в ваш .env файл
4. Запуск проекта
   ```
   npm run dev
   ```
5. Необязательный шаг
   В случае если токен нерабочий, в .env.default есть запасной, скопируйте его и замените ключ.

## Стек технологий

- TypeScript
- React
- React Router
- SCSS modules
- VK UI
- TanStack Query
- Axios
